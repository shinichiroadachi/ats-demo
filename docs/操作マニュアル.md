# 採用管理システム デモ 操作マニュアル

## はじめに
本システムは、応募者を「応募→内定」までの流れで一元管理する採用管理ツールである。画面は左側のメニューから切り替える。

> ※ デモ環境である。データはサンプルで、メール通知・日程調整・ログイン認証・求人媒体連携は含まれない。

## 選考ステージ
応募者は次の流れで進む。

**応募 → 書類選考 → 一次面接 → 最終面接 → 内定 → 入社**

※ 見送る場合は「不採用」、辞退された場合は「辞退」に移す。

---

## 1. ダッシュボード（全体状況）
採用活動の状況をひと目で確認する画面である。

- **上部の数値カード**：総応募数(今月)／選考中人数／内定数(今月)／平均選考日数
- **経路別応募数**：応募がどの経路（求人媒体・人材紹介・リファラル・直接応募）から来たか
- **ステージ別人数**：各選考ステージに何人いるか
- **直近の応募者**：最近の応募者一覧。「パイプラインで見る →」で次の画面へ移動する

## 2. パイプライン（最重要）
応募者をカード形式で管理し、**ドラッグ＆ドロップでステージを移動**できる画面である。

- **カードを移動**：候補者カードをつかんで、隣の列（次のステージ）へドラッグする
- **詳細を見る**：カードをクリックすると右側に詳細パネルが開く
- **辞退／不採用**：カード右上の「⋮」メニューから「辞退にする」「不採用にする」を選ぶ
- **絞り込み**：右上の「すべての職種」から職種を選ぶと、その求人の応募者だけ表示される

## 3. 候補者（一覧）
全応募者を表形式で確認する画面である（不採用・辞退も含む）。

- **絞り込み**：右上で「職種」「ステージ」を選んで一覧を絞る
- **詳細を見る**：行をクリックすると詳細パネルが開く

### 候補者詳細パネル（パイプライン・候補者から開く）
- **基本情報**：応募職種・メール・電話・応募日・経路
- **添付ファイル**：履歴書ファイル名
- **選考履歴**：これまでのステージ変更の記録
- **メモ**：面接所感などを入力し「メモを追加」で保存する

## 4. 求人（募集管理）
募集中の求人を管理する画面である。

- **一覧**：職種名・雇用形態・公開ステータス（公開中／停止）・候補者数・作成日
- **新規追加**：右上の「＋ 求人を追加」から登録する
- **編集**：職種名をクリックすると内容を編集できる

---

## よくある操作
- **応募者を次の選考に進める** → パイプラインでカードを右の列へドラッグする
- **面接メモを残す** → 候補者をクリック → メモ欄に入力 → 「メモを追加」
- **新しい募集を出す** → 求人 → 「＋ 求人を追加」
